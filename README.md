# UserAgentProvider
Random User Agent string provider for Java.

## Maven dependency:

```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency>
    <groupId>com.github.mdebenito</groupId>
    <artifactId>UserAgentProvider</artifactId>
    <version>master-SNAPSHOT</version>
</dependency>
```
